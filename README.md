# 📨 Email Scheduler Server

매일 정해진 시간에 이메일을 자동 발송하는 Java 기반 스케줄러 서버를 구현합니다.  
스케줄링, 메시지 큐, 비동기 처리, 동시성 제어 등의 개념을 직접 경험하며 백엔드 서버의 동작 원리를 이해하는 것이 목표입니다.

---

## 프로젝트 선정 배경

평소 자주 이용하는 뉴스레터나 매일메일 서비스가  
"정해진 시간에 수많은 사용자에게 동시에" 메일을 발송하는 방식을 보며 궁금증이 생겼습니다.

대규모 사용자에게 정시 발송되는 알림/메일 시스템이 **어떤 구조로 동작하는지**,  
이를 가능하게 하는 관련 개념들을 학습하여 직접 구현해보며 동작원리를 깊이 이해하는 것을 목표로 합니다.

---

## 프로젝트 목표

- Spring Boot 기반의 스케줄링 서버 설계 및 구현
- RabbitMQ를 이용한 메시지 큐 기반의 비동기 처리 구조 학습
- 주기적인 작업 수행(`@Scheduled`)을 통한 자동화 로직 이해
- 멱등성, 동시성 제어를 통한 안정적 메시지 처리 실습
- Docker Compose를 활용한 로컬 인프라 환경 구성 (`RabbitMQ`, `DB` 등)
- 테스트 도구를 사용해 서버 부하 및 동시성 테스트 진행
